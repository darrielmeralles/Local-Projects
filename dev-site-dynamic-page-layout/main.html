Desciption

This widget typically provides a dynamic and user-friendly interface for displaying various geographic data, such as state boundaries, cities, and more. Users can often zoom in and out, hover over states for additional information, and click on specific regions to access relevant data or perform actions. These widgets are commonly used on websites, dashboards, or applications to provide an engaging and informative way to present and explore geographical information related to the United States

Features

  • Tooltip Trigger

    Tooltip can be trigger via “CLICK” or “HOVER”

  • Predefine States

    States are already added on the dropdown which you can easily selects what are the states are they going to highlight on the map

  • Selected State Color

    customizable state color using your preferred color(supports: rgba, rgb, hex, and color name)

  • Video Tooltip Content

    this widgets supports YOUTUBE and VIMEO as source of the video.

  • Customizable Tooltip box and Contents

    this widget also allows the user to customize the Tooltip box and content using their preferred color scheme and font style

  • Hide Content if Emty

    it automatically hides the contents if the fields is empty

  • Display State Name

    toggle features to Hide and Show State name in general
